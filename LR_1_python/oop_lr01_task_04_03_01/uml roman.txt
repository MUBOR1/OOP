Класс Roman представляет римское число и предоставляет методы для работы с ним.

Атрибуты класса:

_arabic - целочисленное значение римского числа.
Методы класса:

__init__(value: Union[int, str]) - конструктор класса, принимает на вход либо целое число, либо строку с римским числом. Инициализирует атрибут _arabic с числовым значением римского числа.
__add__(other: Union[Roman, int]) - перегрузка оператора сложения. Позволяет складывать римские числа между собой или римское число с целым числом.
__sub__(other: Union[Roman, int]) - перегрузка оператора вычитания. Позволяет вычитать римские числа или римское число из целого числа.
__mul__(other: Union[Roman, int]) - перегрузка оператора умножения. Позволяет умножать римские числа между собой или римское число на целое число.
__floordiv__(other: Union[Roman, int]) - перегрузка оператора целочисленного деления. Позволяет делить римские числа или римское число на целое число.
__truediv__(other: Union[Roman, int]) - перегрузка оператора деления. Позволяет делить римские числа или римское число на целое число.
__str__() - перегрузка оператора вывода в строку. Возвращает строку с римским числом.
to_arabic(roman: str) -> int - статический метод класса, преобразует римское число в арабское (целое) число.
to_roman(arabic: int) -> str - статический метод класса, преобразует арабское (целое) число в римское число.
